GitHub

GitHub

Jednym z celi warsztatu jest pokazanie uczestnikom w jaki sposób publikować dokonane w projekcie zmiany aby były dostępne one dla innych członków zespołu. Do tego celu wykorzystamy darmowy portal GitHub.

Na potrzeby warsztatu uczestnik musi założyć i zweryfikować swoje konto w portalu GitHub (instrukcja dostępna jest tutaj).

SSH

Po pomyślnym zweryfikowaniu konta w portalu GitHub należy skonfigurawać dostęp do portalu za pomocą protokołu SSH. Pełna instrukcja jak tego dokonać dostępna jest w dokumentacji GitHub.

Prework (3/3)

Aby zweryfikować czy nasze połączenie za pomocą protokołu SSH zostało skonfigurowane poprawnie należy w wierszu poleceń wykonać następujące polecenie:

ssh -T git@github.com
